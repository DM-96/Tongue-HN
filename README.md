# Tongue Hacker News App

---

## Descrizione
Tongue Hacker News App è un'applicazione web che permette di consultare le ultime notizie e le top news provenienti da [Hacker News](https://news.ycombinator.com/).  
L'app fornisce un’interfaccia semplice e moderna, con la possibilità di caricare le news a batch di 10, visualizzando titolo, link e data di pubblicazione.

L'app è sviluppata in **HTML, CSS e JavaScript puro**, senza framework come React, Angular o Vue.

---

## Caratteristiche
- Visualizzazione delle **Ultime** e **Top** news.
- Paginazione con pulsante **Load More**.
- Layout moderno e responsive con navbar e footer.
- Facile da estendere o personalizzare.

---

## Requisiti
- Browser moderno (Chrome, Firefox, Edge, Safari)
- Connessione internet per accedere alle API di Hacker News
